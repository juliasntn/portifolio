SMART GLASS

Este projeto visa monitorar e analisar as paradas de máquinas, fornecendo os principais indicadores de desempenho, como MTBF (Tempo Médio Entre Falhas) e MTTR (Tempo Médio Para Reparo). A plataforma também permite a comparação entre diferentes equipamentos, facilitando a identificação de oportunidades de melhoria e a otimização da manutenção.

Tecnologias Utilizadas

Fast API Docker SQL Server HTML JS TailWind OPCUA

Informações do Projeto

O sistema coleta dados de parada de máquina diretamente do CLP através do protocolo OPC UA, fornecendo em tempo real o status e o tempo de inatividade de cada equipamento.

Este projeto teve como principal objetivo substituir duas plataformas anteriores (G-Vidros e cálculo de MTBF/MTTR via SAP) que apresentavam baixa precisão devido à dependência de alimentação manual de dados, eliminando assim as falhas humanas no input das informações.

Principais funcionalidades: verificação de tempo de parada, classificação de tipo de falha, identificação de paradas programadas, integração com 5 Porquês, comparação de indicadores entre máquinas e cálculo automático de MTBF, MTTR e percentual de paradas programadas.

Resultado financeiro: economia de R$ 120.000,00 que seria gasto com solução externa, além da vantagem do lançamento automático de paradas, eliminando a dependência de input manual.
