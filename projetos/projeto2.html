Certificados Projetos Sobre

SMART MOLD

Plataforma digital para PTP de moldes que substitui formulários em papel e sistemas terceiros, oferecendo interface web e mobile para maior eficiência operacional e autonomia no armazenamento de dados.

Tecnologias Utilizadas

Fast API Grafana Docker SQL Server HTML CSS JS

Informações do Projeto

Coleta em Tempo Real

Coleta de dados em tempo real de sensores industriais com processamento via Python e armazenamento em banco SQL.

Containerização

Utilização de Docker para containerização e Grafana para visualização de painéis dinâmicos e alertas de performance.

Solução Escalável

Foco na criação de solução escalável e de fácil manutenção, com segurança reforçada e automação completa.

Infraestrutura Robusta

Investimento em servidor PowerEdge R260 para garantir redundância e alta disponibilidade do sistema.

Contexto e Problemas Anteriores

Processo Manual Ineficiente

Medições manuais com anotações em papel e lançamento em plataforma terceira ao final do dia.

Falhas Críticas

Perda frequente de papéis com anotações e perda de dados na plataforma externa devido a sobrecargas do sistema.

Solução Desenvolvida - Funcionalidades

  • Banco de dados redundante (armazenamento em dois servidores diferentes)
  • Backup automático de dados
  • Controle de acesso por login e senha
  • Área exclusiva para modificações do PTP com login específico para qualidade
  • Área para comentários quando houver parâmetros fora do PTP
  • Sistema de confirmação de valores
  • Área para verificação de histórico completo
  • Módulo para verificação de frequência analítica

Investimento em Infraestrutura

Servidor PowerEdge R260: R$ 13.669,84

Objetivo: Garantir redundância do banco de dados considerando a criticidade das informações para os processos.
