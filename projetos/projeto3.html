VIRTUALIZAÇÃO DE SUPERVISÓRIOS

A finalidade do projeto é virtualizar os sistemas supervisórios e SCADA, implementar backups automáticos utilizando o Veeam e atualizar os sistemas legados.

Tecnologias Utilizadas

HCI VEEAM HYPER-V DELL V10 Factory Talk

Informações do Projeto

A virtualização das máquinas surgiu para resolver um problema crítico na área, onde dois sistemas supervisórios responsáveis pelo controle automático do processo foram corrompidos e não possuíam backups. Esta situação motivou a implementação de uma infraestrutura HCI com virtualização via Hyper-V, aproveitando a oportunidade para atualizar os sistemas supervisórios da versão FT 4 para FT 15.

Implementamos o VEEAM para backups automáticos e utilizamos Dell V10, equipamentos que seriam descartados, para disponibilizar acesso RDP à equipe. A solução proporcionou uma economia de R$ 400.000 em comparação com computadores industriais convencionais, ou R$ 80.000 considerando a alternativa com QCM1250.

Além dos ganhos financeiros, o projeto eliminou falhas humanas nos backups e reduziu custos com atualizações de software para FT 15, criando uma infraestrutura mais robusta e confiável para os sistemas de automação.
