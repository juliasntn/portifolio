CRIAÇÃO DE SERVIDOR

Este projeto foi desenvolvido com o objetivo de otimizar o processo de monitoramento e análise de dados em ambientes industriais, integrando sensores IoT e dashboards inteligentes.

Tecnologias Utilizadas

Python Grafana Docker SQL Node-RED

Informações do Projeto

O sistema desenvolvido realiza coleta de dados em tempo real de sensores industriais, processando as informações via Python e armazenando em um banco de dados SQL. Utilizando Docker para containerização e Grafana para visualização, o projeto oferece painéis dinâmicos e alertas de performance.

O foco principal foi criar uma solução escalável e fácil de manter, com segurança reforçada e automação de processos de coleta e análise.
